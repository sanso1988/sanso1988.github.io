1. Scalars, Vectors, Matrices and Tensors

In computer science, they regularly call scalar, vector, matrix and tensor are several types of objects. Scalar is just a single number. Vector is an array of numbers. Matrix is a 2-D array of numbers. Tensors is a N-D array of numbers.

Matrix is the core of linear algebra what represents the linear transformation. We primarily impose several opertaions on matrix object.

Transpose means getting the reflection of matrix across the diagonal line. We denote the transpose of a matrix A as AT, the entries of A and AT have the relationship as defined below.

(AT)i,j=Aj,i

add means add corresponding elements of two matrix with the same shape.

C=A+B where Ci,j=Ai,j+Bi,j

2. Multiplying Operation

Multiplying is crucial operation in linear algebra, actually it means transformation of vectors. Multiply of two matrices defined as below.

C=AB where Ci,j=∑kAi,k∗Bk,j

As defined, we could the number of A's columns needed to be the same as the number of B's rows. Vector could be treated as matrix with one column, the multiply of matrix and vector looks like this.

v∗=Av where vi∗=∑kAi,k∗vk

If we see the vector as a object which has a direction and length. Multiplying matrix A means impose a linear transformation on v, v∗ maybe has a new direction and length.

Multiplying operation also called as produc operation has some good properties like distributive, associative.

A(B+C)=AB+AC

A(BC)=(AB)C

And Transpose of a matrix product has the simple form.

(AB)T=BTAT

With production operation, a system of linear equation could be written down in simple form.

Ax=b

3. Identity and Inverse Matrices

Inverse matrix is very crucial to solve the system of linear equation and useful for many scientific area. For illustrating inverse matrix, we need to introduce identity matrix primarily.

Identity matrix means it don't change any vector when we product the matrix with the vector. It's easy to prove the identity matrix must have the structure of all the entries along the dragonal are 1, others are 0, and be square matrix.

Inx=x,∀x∈Rn

Inverse matrix of A is denoted as A−1 which multiply A equals identity In.

A−1A=In

If we have the inverse matrix, we could solve the system of linear equation easily.

Ax=b

A−1Ax=A−1b

x=A−1b

Actually inverse matrix is merely theoretical tool, it costs too much to get the precision. In practice, many software use iteration algorithms to get the enough accurate solution of the system.

4. Linear Dependance and Span

Sometimes the system of linear equation has more than one solution, let us look into what the solutions like. If x,y are both solution of system, it's obvious ax+(1−a)y,∀a∈R is the solution too. Extensily, we call the linear add of several vectors linear combination.

∑civi is a linear combination of vi,i=1,2,3,...

The span of vi means the set of all vectors (points) could be obtained by linear combination of vi.

If we treat the system of equation as linear combination of the columns of A to reach point b, whether solution exist depends on whether b is in the span of columns.

Ax=b

∑A:,ixi=b

If we think about another problem, what determine whether one or more solution exist. If any column was a linear combination of other columns, there must be more than one solution, cause we could put any part of the column coefficient to the combination which is still the solution.

We call a set of vectors linear dependance as any of vectors can be linear combination of others. Conversely, linear independent.

It can be proved that there must be unique solution or no solution if the column vectors are linear independant.

And let us consider how to guarantee where must be a solution exist. It means the span of the column vectors is the whole space Rn.

So, columns of A is linear independent and whose span is real number space of dimension n could means the solution exists and is only one. In my memory, we call the columns basis as this case.

  • f(x+y)<=f(x)+f(y)

L stands for a famous french mathematician Henri Lebesgue who described this measure in his dissertation in 1902.

6. Special Kinds of Matrices and Vectors

Some special kinds of matrices and vectors are very useful, especially as the basic block to construct the complicated one.

Diagonal matrices has nonzero entries only along the main diagonal. It's easy to figure out the inverse matrix is Diagonal either and the corresponding element is the inverse one of original.

Symmetric matrix is equal to its own transpose. It's obvious that the left-down elements are the mirroring of right-up elements.

Unit vector is a vector with unit norm.

Nonzero vector x and y are orthogonal means xTy=0, if the norm of x and y are both 1, we call them orthonormal.

A square matrix called orthogonal matrix if and only if ATA=AAT=I, it implies A−1=AT.

7. Eigendecomposition

Matrix could be seen as a linear transformation, what's the direction is unchanged under the transformation is important, cause it reveals what's stable.

An eigenvector of a square matrix A is a nonzero vector v such that multiplication by A alters only the scale of v.

Av=λv

The scalar λ is called as corresponding eigenvalue.

If v is eigenvector, any sclar multiply v is eigenvector too, so we usually look only for unit eigenvector.

Suppose that a square matrix A has n linear independent eigenvectors construct the matrix V with corresponding eigenvalues construct the diagonal matrix diag(λ), we could get the eigendecomposition of A.

A=Vdiag(λ)V−1

Not all the square matrix have the eigendecomposition. In my memory, the square matrix need to be full-ranked.

Specifically, every real symmetric matrix can be decomposed into an expression involving orthogonal matrix constructed by eigenvectors and diagonal matrix constructed by eigenvalues.

A=Qdiag(λ)QT

Where Q is an orthogonal matrix constructed by eigenvectors, diag(λ) is diagonal matrix constructed by corresponding eigenvalues.

A symmetric square matrix whose eigenvalues are all positive is called positive definite, are all positive or zero is called positive semidefinite. Likewise, it's easy to define negative definite and negative semidefinite. With the eigendecomposition, it's obvious that for any positive definite or semidefinite matrix A, xTAx>=0,∀x.

8. The Trace Operator

The trace of a matrix is given the sum of all the diagonal entries.

Tr(A)=∑Ai,i

The trace operator makes some concepts written down easily, like Frobenius norm.

||A||F=Tr(AAT)

The trace operator is invariant to the transpose operator.

Tr(A)=Tr(AT)
